## What it does

<CardGrid>
  <Card title="Finds more than 'save image as'" icon="magnifier">
    Lazy-loaded images (`data-*`), `srcset`, `picture`, CSS backgrounds, gallery links, `noscript` fallbacks, and `video` / `audio` sources.
  </Card>
  <Card title="Original quality" icon="rocket">
    De-proxies wrapped URLs and rewrites CDN thumbnails to full size. Opt-in "resolve originals" fetches the exact highest-resolution file for supported hosts.
  </Card>
  <Card title="Filter, search & select" icon="setting">
    Filter by kind / format / size, search by name·alt·URL, sort, shift-select ranges, then download one item, a selection, or a single ZIP.
  </Card>
  <Card title="Reliable downloads" icon="approve-check">
    A resilient queue tracks each file, retries transient failures, resumes after the popup closes, and recovers hotlink-protected files with a referer retry.
  </Card>
  <Card title="Works on the sites you use" icon="puzzle">
    Original-quality resolvers for dozens of platforms plus 60+ CDN families — see the [coverage matrix](/media-bulk-downloads/benchmark/coverage-matrix/).
  </Card>
  <Card title="Private by design" icon="approve-check">
    Network-free by default — it reads only what the page already loaded. No accounts, no analytics, no servers; settings and history never leave your device.
  </Card>
</CardGrid>
